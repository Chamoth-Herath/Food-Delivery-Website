# Food Delivery Management System (MERN Stack)

## Overview
A full-stack food ordering web application that allows customers to register, browse food items, manage cart, make secure online payments, and track orders in real time.

The system includes separate **User and Admin panels** for managing food items, orders, and customer activities.

## Features
- User registration and authentication
- Food browsing and search functionality
- Cart management system
- Secure online payment integration
- Order placement and tracking
- Admin panel for managing menu and orders
- Real-time order status updates

## Technologies Used
- React JS (Frontend)
- Node JS (Backend)
- Express JS (Server framework)
- MongoDB (Database)
- Stripe (Payment gateway)
- Clerk (Authentication and user management)

## Testing & Quality Assurance
- End-to-end system testing
- API testing for backend services
- Payment flow validation (Stripe integration)
- Cart and order functionality testing
- UI/UX testing across user flows
- Reported and documented frontend and backend defects

## Project Type
Full-Stack MERN Project
